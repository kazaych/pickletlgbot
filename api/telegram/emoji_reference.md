## Часто используемые эмодзи

### Действия
- ➕ `➕` - Добавить/Создать
- ➖ `➖` - Удалить
- ✏️ `✏️` - Редактировать
- ✅ `✅` - Успешно/Готово
- ❌ `❌` - Ошибка/Отмена
- 🔄 `🔄` - Обновить
- 🔍 `🔍` - Поиск
- 📝 `📝` - Записать/Создать

## Как использовать

1. **Скопировать напрямую** из этого файла или из любого источника эмодзи
2. **Использовать палитру эмодзи**:
   - macOS: `Cmd + Ctrl + Space`
   - Windows: `Win + .`
   - Linux: зависит от дистрибутива
3. **Онлайн ресурсы**:
   - https://emojipedia.org/
   - https://unicode.org/emoji/charts/
   - https://getemoji.com/
